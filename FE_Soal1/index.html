Tentang Saya

Perkenalkan nama Saya M. Ilham Syafik saya sekarang sedang berkuliah sebagai mahasiswa Semester 1 jurusan Teknik Informatika di Universitas Sriwijaya. Saya lahir di Palembang 10 Februari 2005, dan sekarang alhamdulillah masih tinggal di Palembang bersama kerluarga. Hobi saya adalah bermain game, menonton anime serta membaca manga. Saya biasanya menghabiskan waktu dengan membaca manga ataupun bermain game. Kelebihan saya yaitu saya bisa dengan mudah beradaptasi dengan sesuatu yang baru, kekurangan saya mungkin saya tidak begitu teliti.

Riwayat Pendidikan :

- SD Kartika 2 Palembang

- SMP Muhammadiyah 4 Palembang

- MAN Insan Cendekia OKI

Dan sekarang saya sedang berkuliah di Universitas Sriwijaya Teknik Informatika S1. Senang bisa memperkenalkan diri saya, semoga hari anda menyenangkan dan saya doakan yang terbaik untuk anda.
